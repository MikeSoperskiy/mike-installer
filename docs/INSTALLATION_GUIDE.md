# Руководство по установке

## Требования

- Windows 10/11
- Node.js 18 или новее
- Winget (обычно предустановлен в Windows 11)

## Установка Node.js

Если Node.js еще не установлен:

1. Перейдите на https://nodejs.org/
2. Скачайте LTS версию
3. Установите с настройками по умолчанию

## Проверка Winget

Откройте PowerShell и выполните:

```powershell
winget --version
```

Если команда не найдена, установите App Installer из Microsoft Store.

## Установка Mike Installer

### Из исходного кода

```bash
# Клонирование
git clone https://github.com/MikeSoperskiy/mike-installer.git
cd mike-installer

# Установка зависимостей
npm install

# Запуск
npm start
```

### Сборка исполняемого файла

```bash
# Для Windows
npm run build:win

# Готовый инсталлятор будет в папке dist/
```

## Использование

1. Запустите Mike Installer
2. Выберите необходимые программы
3. Нажмите "Установить выбранное"
4. Дождитесь завершения установки

## Добавление пользовательских программ

1. Нажмите "➕ Добавить программу"
2. Введите данные:
   - Название
   - Winget ID (найти через `winget search`)
   - Категория
   - Описание
3. Нажмите "Добавить"

## Решение проблем

### Программа не устанавливается

- Запустите приложение от имени администратора
- Проверьте интернет-соединение
- Обновите winget: `winget upgrade --all`

### Winget не находит программу

Проверьте правильность Winget ID:

```bash
winget search "название"
```

### Ошибки сборки

Убедитесь, что:
- Node.js версии 18+
- Все зависимости установлены (`npm install`)
- Достаточно места на диске
